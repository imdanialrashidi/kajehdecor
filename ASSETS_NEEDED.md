# Assets Needed / فایل‌های مورد نیاز

## لوگو
- [ ] لوگوی اصلی کاژه دکور (SVG یا PNG با کیفیت بالا)
- [ ] لوگوی favicon (32x32, 16x16)

## تصاویر پروژه‌ها
- [ ] تصاویر واقعی آشپزخانه مدرن مهرانشهر (قبل و بعد)
- [ ] تصاویر بازسازی آپارتمان تهران (قبل و بعد)
- [ ] تصاویر دفتر کار مدرن کرج (قبل و بعد)
- [ ] تصاویر رستوران سنتی-مدرن تهران
- [ ] تصاویر بازسازی ویلا البرز (قبل و بعد)

## تصاویر سایت
- [ ] تصویر hero اصلی (آشپزخانه یا نشیمن مدرن)
- [ ] تصویر hero ثانویه
- [ ] تصاویر مقالات وبلاگ (4 عدد)
- [ ] تصویر OG default برای شبکه‌های اجتماعی (1200x630)

## اطلاعات تماس نهایی
- [ ] شماره واتساپ نهایی (فعلاً: 09934202034)
- [ ] آدرس نهایی (فعلاً: تهران، بزرگراه جناح، خیابان طاهریان، پلاک ۳)
- [ ] ساعات کاری نهایی (فعلاً: شنبه تا پنجشنبه ۹-۲۰)
- [ ] آیا ایمیل هم وجود دارد؟

## محتوا
- [ ] متن نهایی نظرات مشتریان (فعلاً از سایت فعلی استخراج شده)
- [ ] نظرات بیشتر اگر موجود است
- [ ] تعداد واقعی پروژه‌های انجام شده
- [ ] سال تأسیس یا تجربه کاری
- [ ] محتوای نهایی مقالات وبلاگ (فعلاً دمو است)

## متریال
- [ ] آیا گواهینامه یا جایزه‌ای وجود دارد؟
- [ ] آیا همکاری با برند خاصی وجود دارد؟
- [ ] عکس تیم یا خانم محمدی (طراح)
